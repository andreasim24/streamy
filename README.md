This is video-sharing app that allows users to create and share videos on any topic inspired by TikTok.

## Features

This repository is packed with:

- ⚡️ Next.js 12
- ⚛️ React 18
- ✨ TypeScript
- 💨 Tailwind CSS 3
- 📦 Sanity
